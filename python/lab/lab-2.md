# 基础练习-2

### 目标
写一个程序，实现模拟用户登录的过程：
```sh
C:\python\lab>python lab-2.py
******************************
*欢迎来到xx系统，请先验证身份*
******************************

用户名：zhangsan
密码：123
用户名或密码错!
用户名：zhangsan
密码：123456
登录成功！
```

### 要求
1. 程序内置一组帐号密码
2. 显示欢迎信息
3. 通过用户输入，验证帐号和密码
4. 连续输入3次错误的信息，显示提示信息，终止程序
5. 帐号和密码与预置的信息一致，显示登录成功信息，结束程序
6. 程序保存为lab-2.py
