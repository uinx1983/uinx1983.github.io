### Hello World
> Python的基本语法更简单（说明自带的函数功能更丰富）

C语言
```c
printf("Hello World!\n");
```
Python
```python
print("你好 世界!")
```


### <a id="var" />变量
> Python的变量不用声明类型

C语言
```c
int i=0;
printf("integer i is %i!\n",i);
```
Python
```python
i=0
print(f"integer i is {i}!")
```
